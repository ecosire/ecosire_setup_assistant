Odoo Setup Assistant

Your comprehensive diagnostic and configuration toolkit for Odoo environments.

The Odoo Setup Assistant is an essential module for developers, administrators, and consultants. It streamlines environment verification, dependency management, and configuration analysis, ensuring your Odoo instance runs smoothly and efficiently.

‚ú® Key Features

üêç Python & System Dependencies Check

Automatically scans for required Python libraries and system-level executables (like wkhtmltopdf, psql), highlighting any missing components critical for Odoo's operation and specific module functionalities.

üì¶ Addon Python Requirements Management

Scans all custom and community addons for requirements.txt files. Lists missing Python packages and provides a one-click option to install them directly into your Odoo environment.

üíæ Database Health Analysis

Verifies database connection parameters from odoo.conf, attempts direct PostgreSQL server connection, and checks Odoo's current database session to ensure robust connectivity and proper configuration.

‚öôÔ∏è Odoo.conf Deep Scan

Analyzes your odoo.conf file for critical path validations (addons_path, data_dir), performance settings (workers, limits), security configurations (admin_passwd, list_db), and common misconfigurations or typos.

üìÑ Log File Analysis & Configuration

Provides tools to fetch, filter (by level, keyword), and display recent log entries. Includes diagnostic hints for common errors. Allows setting a custom log file path for analysis, overriding the odoo.conf default for targeted troubleshooting.

üõ†Ô∏è Automated Dependency Installation

Offers a convenient action to attempt automatic installation of identified missing Python libraries and system dependencies directly from the Odoo interface (requires appropriate server permissions).

üîÑ Odoo Service Restart

Provides a direct button within the assistant to safely initiate a restart of the Odoo service, useful after installing new dependencies or applying configuration changes (requires appropriate server permissions).

üìä User-Friendly Interface

All checks and actions are available within a clean, tabbed Odoo interface, making it easy to navigate and interpret results. Supports opening in the main Odoo window for a full-view experience.

üîó GitHub Repository Management

Configure GitHub repositories (public or private with PAT) to easily clone or pull module updates directly into your Odoo addons path from the interface. Automate code deployment for smoother development workflows.

üéØ Who is it for?

  • Odoo Developers needing rapid environment validation.
  • System Administrators maintaining Odoo instances.
  • Technical Consultants provisioning or troubleshooting client setups.
  • Teams managing multiple Odoo versions and configurations.

üìÇ Installation

1. Clone or download the odoo_setup_assistant module.

2. Place it into your Odoo addons directory.

3. Restart your Odoo server.

4. Navigate to 'Apps' in Odoo, search for "Odoo Setup Assistant", and click 'Activate'.

üìå Pro Tip

Regularly use the Odoo Setup Assistant after server updates, new module installations, or configuration changes to proactively identify potential issues. Integrate checks into your CI/CD pipeline for automated environment validation.
