Odoo Setup Assistant

Optimize, validate, and automate your Odoo environment with ease.

The Odoo Setup Assistant empowers developers, consultants, and sysadmins with powerful diagnostic tools to streamline your Odoo deployment. Easily detect configuration issues, manage dependencies, and validate your environment — all from within Odoo.

✨ Comprehensive Features

🐍 Python & System Requirements

Identify missing Python libraries and essential binaries like wkhtmltopdf and psql to ensure your Odoo runs without errors.

📦 Auto-Detect Addon Dependencies

Scan all addons for requirements.txt files and highlight any unmet dependencies. One-click install available.

💾 Database Connectivity Report

Validate your PostgreSQL connection using credentials from odoo.conf and test live Odoo DB session health.

⚙️ Config File Scanner

Audit your odoo.conf for critical paths, tuning parameters, and potential misconfigurations including security flaws.

📄 Log Analyzer

View recent log entries, filter by keyword or log level, and get actionable insights. Supports overriding log file paths for targeted debugging.

🛠️ One-Click Fixes

Trigger automated installation of missing packages and tools directly from the UI. Requires sufficient server-level permissions.

🔄 Restart Odoo Instantly

Initiate a safe restart of the Odoo service right from the interface — ideal after changes or installations. (Requires sudo access)

📊 Polished Interface

Enjoy an intuitive, tabbed UI that makes diagnostics easy and accessible within Odoo. Fully responsive and easy to use.

🔗 GitHub Sync Management

Connect and manage GitHub repositories to sync or update your custom addons, even from private sources.
